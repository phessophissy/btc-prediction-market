# Frontend Environment

Supported frontend environment variables:

- `NEXT_PUBLIC_NETWORK`
- `NEXT_PUBLIC_CONTRACT_ADDRESS`
- `NEXT_PUBLIC_CONTRACT_NAME`

Defaults are provided for the deployed mainnet V3 contract, but a local or alternate deployment can override them without source edits.

## Fee Configuration

`NEXT_PUBLIC_DEFAULT_FEE_BPS` – basis points for the default fee tier (default: 300).
