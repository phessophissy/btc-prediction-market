# Frontend Theme System

The current UI uses a colorful dark theme defined in `frontend/src/app/globals.css`.

## Design choices

- Fraunces for headings
- Space Grotesk for body copy
- layered gradients and glow effects
- shared component classes such as `.card`, `.hero-panel`, and `.btn-primary`

## Maintenance note

Prefer reusing the shared CSS classes before introducing per-page one-off utility stacks.
